# 个人博客项目
项目使用了前后端分离的架构,这个项目仅仅为前台项目,后台项目提交到了另一个仓库地址
生成环境中
## 1.下载依赖
```
npm install
```

### 2.启动项目(注:本项目仅为前台项目后台放置在另个仓库)
```
npm run serve
```

### 3.打包项目
```
npm run build
```
